<h2 style="text-align: center;"> <span style={{ color: 'purple', fontWeight: 'bold' }}> Δημιουργία repository </span> </h2>

Για να δημιουργήσουμε ένα νέο repository στον υπολογιστή μας, μπορούμε να ακολουθήσουμε δύο τρόπους.

1. Να δημιουργήσουμε ένα νέο κενό repository στο GitHub και να το κάνουμε clone στον υπολογιστή μας.
2. Να δημιουργήσουμε ένα νέο κενό repository στον υπολογιστή μας και να το συνδέσουμε με ένα υπάρχον repository στο GitHub.

<h3 style="text-align: left;"> <span style={{ color: '#699BE0'}}> Δημιουργία κενού repository στο GitHub </span> </h3>

1. Αρχικά, μπαίνουμε στην [ιστοσελίδα του GitHub](https://github.com) με τον λογαριασμό που κάναμε στο προηγούμενο μάθημα.
2. Επιλέγουμε το κουμπί `+` πάνω δεξιά και μετά `New repository`.
3. Επιλέγουμε ως owner τον λογαριασμό μας και δίνουμε ένα όνομα στο repository μας.
4. Μπορούμε προαιρετικά να βάλουμε μια περιγραφή για το repository μας.
5. Επιλέγουμε να κάνουμε το repository μας `public` ή `private`.
6. Επιλέγουμε το `Create repository`.
7. Στην επιλογή `Quick setup` επιλέγουμε το `HTTPS` και αντιγράφουμε το link που μας δίνει.
8. Στον υπολογιστή μας και ανοίγουμε το τερματικό.
- Windows: Κάνουμε δεξί κλικ και επιλέγουμε Git Bash
- Mac: Terminal
- Linux: Terminal
9. Γράφουμε `git clone ` και κάνουμε επικόλληση το link που αντιγράψαμε προηγουμένως
10. Αφού πατήσουμε enter, έχει δημιουργηθεί ένας φάκελος, ο οποίος είναι το repository που έχουμε στο GitHub

<h3 style="text-align: left;"> <span style={{ color: '#699BE0'}}> Δημιουργία κενού repository στον υπολογιστή μας και σύνδεση με το GitHub </span> </h3>

1. Δημιουργούμε έναν φάκελο στον υπολογιστή μας, όπου θα βάλουμε το repository μας
2. Ανοίγουμε το τερματικό μέσα στον φάκελο
- Windows: Κάνουμε δεξί κλικ και επιλέγουμε Git Bash
- Mac: Terminal
- Linux: Terminal
3. Τρέχουμε την εντολή `git init`
4. Μπαίνουμε στην [ιστοσελίδα του GitHub](https://github.com) με τον λογαριασμό που κάναμε στο προηγούμενο μάθημα
5. Επιλέγουμε το κουμπί `+` πάνω δεξιά και μετά `New repository`
6. Επιλέγουμε ως owner τον λογαριασμό μας και δίνουμε ένα όνομα στο repository μας
7. Μπορούμε προαιρετικά να βάλουμε μια περιγραφή για το repository μας
8. Επιλέγουμε να κάνουμε το repository μας `public` ή `private`
9. Επιλέγουμε το `Create repository`
10. Στην επιλογή `Quick setup` επιλέγουμε το `HTTPS` και αντιγράφουμε το link που μας δίνει
11. Επιστρέφουμε στο τερματικό και γράφουμε `git remote add origin ` και κάνουμε επικόλληση το link που αντιγράψαμε προηγουμένως

<h2 style="text-align: center;"> <span style={{ color: 'purple', fontWeight: 'bold' }}> Αποθήκευση αλλαγών και commit </span> </h2>

Αφού αρχίσετε να γράφετε τον κώδικά σας στο repository, θα θέλετε να τα κάνετε commit σαν καινούργιες αλλαγές.

Για να προσθέσετε και να κάνετε commit αλλαγές στο Git, ακολουθήστε αυτά τα βήματα:

1. Πριν κάνετε commit στις αλλαγές, είναι σημαντικό να δείτε την κατάσταση των αρχείων σας. Χρησιμοποιήστε την εντολή `git status` στο τερματικό σας για να δείτε τις αλλαγές.

2. Προσθέστε τις αλλαγές που θέλετε να συμπεριλάβετε στο commit χρησιμοποιώντας το `git add`. Μπορείτε να προσθέσετε συγκεκριμένα αρχεία ή όλες τις αλλαγές.
- Για να προσθέσετε συγκεκριμένα αρχεία: `git add όνομα_αρχείου` ή `git add κατάλογος/όνομα_αρχείου`
- Για να προσθέσετε όλες τις αλλαγές: `git add .` (Αυτό τοποθετεί όλα τα τροποποιημένα και νέα αρχεία)

3. Αφού προσθέσατε τις αλλαγές, κάντε commit με ένα μικρό μήνυμα που εξηγεί της αλλαγές χρησιμοποιώντας την εντολή `git commit`.
- `git commit -m "Το μήνυμα σας εδώ"`

4. Τέλος, μπορείτε να στείλετε (push) τις αλλαγές στο GitHub χρησιμοποιώντας `git push`.
- `git push -u origin main`

5. Μπορείτε να ελέγξετε το ιστορικό των commits χρησιμοποιώντας `git log` για να δείτε τις αλλαγές που έχουν κάνει commit.


Ας δούμε ένα παράδειγμα:

```bash
git status
git add όνομα_αρχείου
git commit -m "Το μήνυμά σας εδώ"
git push -u origin main
git log
```
