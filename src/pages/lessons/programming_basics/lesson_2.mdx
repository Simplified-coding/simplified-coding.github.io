<h2 style="text-align: center;"> Μεταβλητές και Τύποι Δεδομένων </h2>

---

<h2 style="text-align: center;"> Τι είναι οι μεταβλητές; </h2>

Οι μεταβλητές είναι "κοντέινερ" που χρησιμοποιούμε στον προγραμματισμό για να αποθηκεύσουμε και να διαχειριστούμε δεδομένα.
Για παράδειγμα, μπορείτε να έχετε μια μεταβλητή με το όνομα "ηλικία" που θα κρατάει την ηλικία ενός ατόμου.
Μπορείτε στη συνέχεια να αλλάξετε τη <τ></τ>ιμή αυτής της μεταβλητής, όταν κάποιος για παράδειγμα γίνεται ένα χρόνο μεγαλύτερος.

Οι μεταβλητές μας επιτρέπουν να αποθηκεύουμε δεδομένα,
να τα επεξεργαζόμαστε και να τα χρησιμοποιούμε στον κώδικα μας.
Κάθε μεταβλητή έχει ένα όνομα που την αναγνωρίζει και μια τιμή που αναπαριστά τα δεδομένα που περιέχει.

---

<h2 style="text-align: center;"> Τι είναι οι τύποι δεδομένων </h2>

Οι τύποι δεδομένων είναι κατηγορίες που καθορίζουν τι μπορούμε να αποθηκεύσουμε στις μεταβλητές μας.
Κάθε τύπος δεδομένων έχει διαφορετικές λειτουργίες.
Ας δούμε τους πιο βασικούς τύπους δεδομένων:

* Ακέραιοι αριθμοί (Integer): Αυτοί είναι οι αριθμοί που δεν έχουν δεκαδικό μέρος. Για παράδειγμα, το 5 είναι ακέραιος αριθμός.
* Δεκαδικοί αριθμοί (Float): Αυτοί είναι οι αριθμοί που έχουν δεκαδικό μέρος. Για παράδειγμα, το 3.14 είναι δεκαδικός αριθμός.
* Κείμενο (String): Χρησιμοποιείται για την αποθήκευση κειμένου, όπως λέξεων ή προτάσεων. Για παράδειγμα, "Γειά σας!" είναι ένα string.
* Λογική τιμή (Boolean): Αυτοί είναι οι τύποι δεδομένων που αντιπροσωπεύουν την αλήθεια (true) ή το ψέμα (false).
  Χρησιμοποιούνται συνήθως για συγκρίσεις και αποφάσεις στο πρόγραμμα.

Αυτοί είναι μόνο μερικοί από τους βασικούς τύπους δεδομένων.
Κάθε γλώσσα προγραμματισμού μπορεί να έχει περισσότερους τύπους,
ανάλογα με τις ανάγκες της και τις δυνατότητες που προσφέρει.
