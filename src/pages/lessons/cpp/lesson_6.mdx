<h2 style="text-align: center;"> <span style={{ color: '#d35763'}}> While </span> loops </h2>

Το while loop είναι ένα πολύ σημαντικό και χρήσιμο εργαλείο στη C++. Έχει την ίδια χρήση και τρόπο χρήσης 
με την Pyhton, μας επιτρέπει να επαναλαμβάνουμε ένα κομμάτι κώδικα για όσο διάστημα ισχύει μια συνθήκη.

Ας δούμε ένα παράδειγμα το οποίο θα μας τυπώσει αριθμούς από το 0 μέχρι το 10:

```cpp
#include <iostream>

using namespace std;

int main() {
    int i = 0;
    while (i <= 9) {
        cout << i << endl;
        i++;
    }
    return 0;
}
```

Ας αναλύσουμε το πρόγραμμα γραμμή προς γραμμή:

<h3 style="text-align: left;"> 
```cpp
while (i <= 9)
``` 
</h3>
Αρχίζουμε την επανάληψη με την εντολή `while` και τη συνθήκη που θα ελέγχεται σε κάθε επανάληψη.
Η συνθήκη μας λέει πως όσο η μεταβλητή `i` είναι μικρότερη ή ίση του 10, θα εκτελείται ο κώδικας μέσα στο `while`.
<br/>

<h3 style="text-align: left;"> 
```cpp
cout << i << endl;
``` 
</h3>
Τυπώνουμε την τιμή της μεταβλητής `i` και αλλάζουμε σειρά στην οποία θα τυωνουμε.
<br/>

<h3 style="text-align: left;"> 
```cpp
i++;
``` 
</h3>
Αυξάνουμε την τιμή της μεταβλητής `i` κατά 1. Όπως και σε άλλες γλώσσες <br/> προγραμματισμού όταν γράφουμε `++` είναι ισοδύναμο με το `i=i+1`.

<h2 style="text-align: center;"> `Do-while loops` </h2>

Το do-while loop είναι παρόμοιο με το while loop, με τη διαφορά ότι ελέγχει τη συνθήκη στο τέλος της επανάληψης.
Αυτό σημαίνει πως θα εκτελεστεί ο κώδικας μια φορά ανεξάρτητος αν η συνθήκη είναι αληθής ή όχι και μετά θα
συνεχίσει αν η συνθήκη είναι αληθής.

Ας δούμε ένα παράδειγμα στο οποίο θα εκτυπώσουμε ένα `11` διότι ο κώδικας θα τυπώσει το `i` μετά θα του προσθέσει +1.
Μετά από αυτό θα ελέγξει αν το i (=12) είναι μικρότερο ή ίσο με 10 το οποίο είναι ψευδές και θα σταματήσει το πρόγραμμα:

```cpp
#include <iostream>

using namespace std;

int main() {
    int i = 11;
    do {
        cout << i << endl;
        i++;
    } while (i <= 10);
    return 0;
}
```
