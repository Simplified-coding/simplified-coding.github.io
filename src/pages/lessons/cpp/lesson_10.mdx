<h2 style="text-align: center;"> Παραπομπές </h2>

Οι παραπομπές στη C++ μπορούν να χρησιμοποιηθούν για να αναφερθούμε 
σε μια μεταβλητή με ένα όνομα διαφορετικό από το δικό της.

<h3 style="text-align: left;"> Δήλωση παραπομπής </h3>

Η δήλωση παραπομπής γίνεται με τον τελεστή `&` μετά τον τύπο της μεταβλητής.

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
    int x = 5;
    int &y = x;
    y = 10;
    cout << x << endl; // 10
    cout << y << endl; // 10
}
```

### Διεύθυνση μνήμης

Η διεύθυνση μνήμης είναι ένας αριθμός που αναπαριστά τη θέση μιας μεταβλητής
στη μνήμη του υπολογιστή. Κάθε φορά που δημιουργείται μια μεταβλητή, η C++
δεσμεύει μια θέση στη μνήμη για αυτήν και αποθηκεύει την τιμή της εκεί.

Μπορούμε με τον τελεστή `&` να πάρουμε τη διεύθυνση μνήμης μιας μεταβλητής.

#### Παράδειγμα

```cpp
#include <iostream>

using namespace std;

int main() {
    int x = 5;
    cout << &x << endl; // 0x7ffeeb0b0a3c
}
```

Μπορεί να μην έχετε το ίδιο αποτέλεσμα, αλλά η διεύθυνση μνήμης που θα
εμφανιστεί θα είναι κάτι παρόμοιο.

## Δείκτες

Οι δείκτες είναι μεταβλητές που περιέχουν μια διεύθυνση μνήμης.
Οι παραπομπές διεύθυνσης μνήμης δεν αποθηκεύονται κάπου ενώ οι
δείκτες αποθηκεύουν τη διεύθυνση μνήμης σε μια μεταβλητή μέσο των παραπομπών.

### Δήλωση δείκτη

Η δήλωση δείκτη γίνεται με τον τελεστή `*` μετά τον τύπο της μεταβλητής και τέλος το όνομα της μεταβλητής.

```cpp

using namespace std;
