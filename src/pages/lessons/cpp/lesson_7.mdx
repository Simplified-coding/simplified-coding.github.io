<h2 style="text-align: center;"> <span style={{ color: '#d35763'}}> For </span> loops </h2>

Στη C++ τα for loops χρησιμοποιούνται όταν ο αριθμός των επαναλήψεων είναι γνωστός εκ των προτέρων

Για να γράψουμε ένα for loop χρησιμοποιούμε την εξής σύνταξη:
```cpp
for (initialization; condition; increment/decrement) {
    // code
}
```

Παράδειγμα:
```cpp
for (i=0; i<10; i++) {
    // code
}
```

Ας αναλύσουμε το παραπάνω κομμάτι κώδικα:

<h3 style="text-align: left;"> <span style={{ color: '#d35763'}}> initialization </span> </h3>

Η `initialization` είναι μια εντολή που εκτελείται μόνο μια φορά στην αρχή του for loop.
Χρησιμοποιείται για να δημιουργήσουμε μια μεταβλητή που θα χρησιμοποιηθεί μόνο στο for loop.
Για παράδειγμα, μπορούμε να δημιουργήσουμε μια μεταβλητή `i` και να της δώσουμε την αρχική τιμή 0.

<h3 style="text-align: left;"> <span style={{ color: '#d35763'}}> condition </span> </h3>

Η `condition` είναι μια συνθήκη που ελέγχεται στην αρχή κάθε επανάληψης.
Όσο η συνθήκη είναι αληθής, το for loop θα συνεχίσει να εκτελείται.
Όταν η συνθήκη είναι ψευδής, το for loop σταματάει να εκτελείται.
Για παράδειγμα, μπορούμε να ελέγξουμε αν η μεταβλητή `i` είναι μικρότερη από 10.

<h3 style="text-align: left;"> <span style={{ color: '#d35763'}}> increment/decrement </span> </h3>

Η `increment/decrement` είναι μια εντολή που εκτελείται στο τέλος κάθε επανάληψης.
Χρησιμοποιείται για να αυξήσουμε ή να μειώσουμε την τιμή της μεταβλητής που χρησιμοποιήσαμε στην `condition`.
Για παράδειγμα, μπορούμε να αυξήσουμε την τιμή της μεταβλητής `i` κατά 1.

Ας δούμε ένα παράδειγμα χρήσης του for loop:

```c++
#include <iostream>

using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        cout << i << endl;
    }
}
```

Το πρόγραμμα αυτό θα εμφανίσει τους αριθμούς από το 0 μέχρι το 10.
