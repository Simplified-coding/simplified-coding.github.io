<h2 style="text-align: center;"> Συναρτήσεις </h2>

---

Οι συναρτήσεις είναι κομμάτια κώδικα που τρέχουν όταν τα καλούμε. Μπορούμε και να τους δώσουμε 
τιμές (παραμέτρους) οι οποίες θα αντικατασταθούν με ό,τι τους πούμε (δείτε το 2ο και το 4ο 
παράδειγμα). Αυτό έχει ως αποτέλεσμα να μη χρειάζεται να γράφουμε τον ίδιο κώδικα πολλές φορές.

<h2 style="text-align: center;"> Δημιουργία συνάρτησης </h2>

Η C++ μας δίνει τη συνάρτηση main() η οποία είναι τρέχει όταν αρχίσει το πρόγραμμα.
Μπορούμε να φτιάξουμε και τις δικές μας συναρτήσεις, έξω από αυτήν.

Για να φτιάξουμε μια συνάρτηση, γράφουμε `void`, το όνομα της και τις παραμέτρους της. Τώρα ας δούμε μερικά παραδείγματα χωρίς αλλά και με παραμέτρους.

Χωρίς παραμέτρους:

```cpp
#include <iostream>

using namespace std;

void printHello() {
    cout << "Γειά!" << endl;
}

int main() {
    // Κώδικας
}
```

Με παραμέτρους:

```cpp
#include <iostream>

using namespace std;

void printHello(string name) {
    cout << "Γειά " << name << "!" << endl;
}

int main() {
    // Κώδικας
}
```

<h2 style="text-align: center;"> Κλήση συνάρτησης </h2>

Για να καλέσουμε μια συνάρτηση, γράφουμε το όνομά της και τις παραμέτρους της. Τώρα ας δούμε μερικά παραδείγματα χωρίς αλλά και με παραμέτρους.

Χωρίς παραμέτρους:

```cpp
#include <iostream>

using namespace std;

void printHello() {
    cout << "Γειά!" << endl;
}

int main() {
    printHello();
}
```

Με παραμέτρους:

```cpp
#include <iostream>

using namespace std;

void printHello(string name) {
    cout << "Γειά " << name << "!" << endl;
}

int main() {
    printHello("Γιώργο");
}
```
