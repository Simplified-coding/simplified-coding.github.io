<h1 style="text-align: center;"> Μάθημα 9 </h1>

<h2 style="text-align: center;"> Δομές (Structures) </h2>

---

Οι δομές στη C++ είναι ένας τρόπος για να ομαδοποιούμε μεταβλητές που έχουν σχέση μεταξύ τους.
Κάθε μεταβλητή είναι ένα μέλος της δομής.

<h2 style="text-align: center;"> Δήλωση δομής </h2>

Η δήλωση μιας δομής γίνεται με την εντολή `struct`.
Μέσα στα άγκιστρα δηλώνουμε μεταβλητές που θα είναι μέλη της δομής.
Έξω από τα άγκιστρα της δομής, δημιουργούμε μία μεταβλητή για αυτήν.

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
    struct {
        string name;
        int age;
    } information;

    return 0;
}
```

---

<h2 style="text-align: center;"> Πρόσβαση στα μέλη μιας δομής </h2>

Για να έχουμε πρόσβαση στα μέλη μιας δομής γράφουμε τη μεταβλητή της δομής και το όνομα του μέλους.

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
    struct {
        string name;
        int age;
    } information;

    information.name = "Μάριος";
    information.age = 16;

    cout << "Όνομα: " << information.name << endl;
    cout << "Ηλικία: " << information.age << endl;

    return 0;
}
```

---

<h2 style="text-align: center;"> Πολλαπλές μεταβλητές δομής </h2>

Μπορούμε να δημιουργήσουμε πολλαπλές μεταβλητές δομής με την ίδια δομή.
Έτσι, με το παράδειγμα, μπορούμε να έχουμε πολλούς ανθρώπους με το ίδιο όνομα και ηλικία.

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
    struct {
        string name;
        int age;
    } information1, information2;

    information1.name = "Μάριος";
    information1.age = 16;

    information2.name = "Φωτεινή";
    information2.age = 42;

    cout << "Όνομα: " << information1.name << endl;
    cout << "Ηλικία: " << information1.age << endl;

    cout << "Όνομα: " << information2.name << endl;
    cout << "Ηλικία: " << information2.age << endl;

    return 0;
}
```

---

<h2 style="text-align: center;"> Δομές με όνομα </h2>

Μπορούμε να δημιουργήσουμε μια δομή με όνομα.
Έτσι, μπορούμε να δημιουργήσουμε πολλαπλές μεταβλητές δομής με το ίδιο όνομα.

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
    struct information {
        string name;
        int age;
    };

    information information1, information2;

    information1.name = "Μάριος";
    information1.age = 16;

    information2.name = "Φωτεινή";
    information2.age = 42;

    cout << "Όνομα: " << information1.name << endl;
    cout << "Ηλικία: " << information1.age << endl;

    cout << "Όνομα: " << information2.name << endl;
    cout << "Ηλικία: " << information2.age << endl;

    return 0;
}
```
