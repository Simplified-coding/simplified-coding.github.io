<h2 style="text-align: center;"> Πράξεις </h2>

Υπάρχουν 5 βασικές πράξεις στη C++:

| Σύμβολο   | Πράξη                         |
|:---------:|:-----------------------------:|
| +         | Πρόσθεση                      |
| -         | Αφαίρεση                      |
| *         | Πολλαπλασιασμό                |
| /         | Διαίρεση                      |
| %         | Υπόλοιπο ακέραιας διαίρεσης   |

(Είναι σχεδόν ίδιες με την Python εκτός από το γεγονός ότι η ύψωση σε τετράγωνο στην c++ γίνετε με την χρήση συναρτήσεων)

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
    cout << 5 + 2 << endl; // 7
    cout << 5 - 2 << endl; // 3
    cout << 5 * 2 << endl; // 10
    cout << 5 / 2 << endl; // 2
    cout << 5 % 2 << endl; // 1
    return 0;
}
```
<h5 style="text-align: center;"> ( Το υπόλοιπο διαίρεσης είναι το υπόλοιπο μιας ευκλείδειας διαίρεσης. Για παράδειγμα, όταν διαιρούμε το 5 με το 2, το αποτέλεσμα είναι 2 με υπόλοιπο 1. ) </h5>

<h2 style="text-align: center;"> Τελεστές σύγκρισης </h2>

Στη C++ μπορεί να θέλουμε να συγκρίνουμε 2 αριθμούς. Αυτό δεν είναι ιδιαίτερα χρήσιμο σε αυτό το στάδιο, 
αλλά είναι χρήσιμο για την εκτέλεση συνθηκών που θα δούμε σε άλλο μάθημα.

Όταν κάνουμε μια σύγκριση, θα έχουμε αποτέλεσμα 0 ή 1.
Το 0 σημαίνει ότι η σύγκριση δεν ισχύει και 1 σημαίνει ότι η σύγκριση είναι σωστή.

Υπάρχουν 6 τελεστές σύγκρισης στη C++ (ακριβώς ίδιοι με την Python):

| Σύμβολο   | Σχέση                  |
|:---------:|:----------------------:|
| `==`      | Ίσο                    |
| `!=`      | Διαφορετικό από        |
| `>=`      | Μεγαλύτερο ή Ίσο       |
| `<=`      | Μικρότερο ή Ίσο        |
| `>`       | Μεγαλύτερο             |
| `<`       | Μικρότερο              |

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
    cout << (5 < 2) << endl;  // 0
    cout << (5 > 2) << endl;  // 1
    cout << (5 == 2) << endl; // 0
    cout << (5 <= 2) << endl; // 0
    cout << (5 >= 2) << endl; // 1
    cout << (5 != 2) << endl; // 1
    return 0;
}
```

<h2 style="text-align: center;"> Λογικοί τελεστές </h2>

Στη C++ μπορούμε να συνδυάσουμε πολλές συγκρίσεις μεταξύ τους χρησιμοποιώντας λογικούς τελεστές.
Ας δούμε τους λογικούς τελεστές και τι κάνει ο καθένας:

| Τελεστής | Σύμβολο | Περιγραφή                                                                    |
|:--------:|:-------:|:----------------------------------------------------------------------------:|
| Και      | `&&`    | Επιστρέφει 1 αν και μόνο αν και οι 2 συνθήκες είναι αληθείς.                 |
| Ή        | `\|\|`  | Επιστρέφει 1 αν και μόνο αν τουλάχιστον μία από τις 2 συνθήκες είναι αληθής. |
| Όχι      | `!`     | Επιστρέφει 1 αν και μόνο αν η συνθήκη είναι ψευδής.                          |

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
    cout << ((5 < 2) && (5 > 2)) << endl; // 0
    cout << ((5 < 2) || (5 > 2)) << endl; // 1
    cout << !(5 < 2) << endl; // 1
    return 0;
}
```

<h2 style="text-align: center;"> Μαθηματικές συναρτήσεις </h2>

Στη C++ μπορούμε να χρησιμοποιήσουμε μαθηματικές συναρτήσεις για να κάνουμε πολύπλοκους υπολογισμούς.
Αυτές οι συναρτήσεις απαιτούν τη βιβλιοθήκη `cmath`.

<h3 style="text-align: left;"> Προσθήκη βιβλιοθήκης </h3>

Για να προσθέσουμε μια βιβλιοθήκη στο πρόγραμμά μας, χρησιμοποιούμε την εντολή `#include` στο `cmath`όπως και με το `iostream`.

```cpp
#include <iostream>
#include <cmath>

using namespace std;

int main() {
    // Κώδικας
    return 0;
}
```

<h3 style="text-align: left;"> Συναρτήσεις </h3>

Ας δούμε τις συναρτήσεις που παρέχει η βιβλιοθήκη `cmath`:

| Συνάρτηση | Περιγραφή                                                        |
|-----------|------------------------------------------------------------------|
| `abs()`   | Επιστρέφει την απόλυτη τιμή ενός αριθμού.                        |
| `sqrt()`  | Επιστρέφει την τετραγωνική ρίζα ενός αριθμού.                    |
| `pow()`   | Επιστρέφει την τιμή ενός αριθμού στην δύναμη ενός άλλου αριθμού. |
| `sin()`   | Επιστρέφει τον ημίνοτο ενός αριθμού.                             |
| `cos()`   | Επιστρέφει τον συνημίτονο ενός αριθμού.                          |
| `tan()`   | Επιστρέφει την εφαπτομένη ενός αριθμού.                          |
| `asin()`  | Επιστρέφει τον αντίστροφο ημίτονο ενός αριθμού.                  |
| `acos()`  | Επιστρέφει τον αντίστροφο συνημίτονο ενός αριθμού.               |
| `atan()`  | Επιστρέφει τον αντίστροφο εφαπτομένη ενός αριθμού.               |
| `log()`   | Επιστρέφει τον φυσικό λογάριθμο ενός αριθμού.                    |

Παράδειγμα:

```cpp
#include <iostream>
#include <cmath>

using namespace std;

int main() {
    cout << abs(-5) << endl;   // 5
    cout << sqrt(25) << endl;  // 5
    cout << pow(5, 2) << endl; // 25
    cout << sin(0) << endl;    // 0
    cout << cos(60) << endl;    // -0.95241298041
    cout << tan(80) << endl;    // 5.6713
    cout << asin(0) << endl;   // 0
    cout << acos(1) << endl;   // 0
    cout << atan(0) << endl;   // 0
    cout << log(2) << endl;    // 0
    return 0;
}
```
