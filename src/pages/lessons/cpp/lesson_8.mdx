<h2 style="text-align: center;"> Πίνακες-Σειρά (Arrays) </h2>

---

Στη C++ οι πίνακες είναι ένας τύπος δεδομένου ο οποίος μας επιτρέπει να αποθηκεύσουμε πολλές τιμές σε μία μεταβλητή. 
Αυτό μας διευκολίνει διότι αντί να φτιάχνουμε πολλές μεταβλητές για πολλές τιμές, κείμενα, κτλ μπορούμε να αλλάζουμε
συνεχώς την τιμή μία και μοναδικής μεταβλητής.

<h2 style="text-align: center;"> Δήλωση array </h2>

Για να φτιάξουμε έναν πίνακα θα γράψουμε τον τύπο των τιμών που θα βάλουμε στον πίνακα και το όνομα της μεταβλητής, μέσα σε αγκύλες 
θα γράψουμε πόσα στοιχεία θα έχει ο πίνακας, θα βάλουμε μέσα σε άγκιστρα τα στοιχεία του πίνακα χωρισμένα με κόμμα.

Ας δούμε ένα παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
	int ages[3] = {15, 32, 47};
	return 0;
}
```

---

<h2 style="text-align: center;"> Δήλωση πίνακα και χωρίς αριθμό στοιχείων </h2>

Στην C++ αν θέλουμε, μπορούμε και να δηλώσουμε έναν πίνακα χωρίς να πούμε τον αριθμό των στοιχείων που θα έχει.
Μπορούμε να το κάνουμε αυτό, αρκεί απλά να μη γράψουμε κάποιο νούμερο μέσα στην αγκύλη.

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
	int ages[] = {15, 32, 47};
	return 0;
}
```

---

<h2 style="text-align: center;"> Πρόσβαση στοιχείων πίνακα </h2>

Αφού δηλώσουμε τον πίνακα, μπορούμε να πάρουμε πρόσβαση σε κάποιο από τα στοιχεία του.
Για να το κάνουμε αυτό, βάζουμε το όνομα του πίνακα και μέσα σε αγκύλες τον αριθμό του στοιχείου.
Δηλαδή στον παραπάνω πίνακα το νούμερο του στοιχείου 15 είναι το 0. Του 32 είναι το 1
και του 47 το 2.

**Προσοχή!** Η αρίθμηση των στοιχείων ξεκινά από το 0 και όχι από το 1 ακριβώς όπως και στην Python.

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
	int ages[3] = {15, 32, 47};
  cout << ages[1];
	return 0;
}
```

Αυτό το πρόγραμμα θα γράψει στην οθόνη του χρήστη τον αριθμό 32.

---

<h2 style="text-align: center;"> Αλλαγή στοιχείων πίνακα </h2>

Για να αλλάξουμε ένα στοιχείο πίνακα, γράφουμε το όνομα του πίνακα και τον αριθμό του στοιχείου
και μετά από το ίσον, βάζουμε τη νέα τιμή του στοιχείου

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
	int ages[3] = {15, 32, 47};
  ages[1] = 46;
  cout << ages[1];
	return 0;
}
```
Έτσι, αυτό το πρόγραμμα θα γράψει στην οθόνη του χρήστη τον αριθμό 36.
