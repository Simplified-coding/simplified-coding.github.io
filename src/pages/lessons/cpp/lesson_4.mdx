<h2 style="text-align: center;"> Οι 3 συνθήκες </h2>

Όπως έχουμε ήδη δει στην c++ υπάρχουν 6 τελεστές σύγκρισης. Οι 3 αυτές συνθήκες σε συνδιασμό με του τελστές μας 
επιτρέπεουν να ελέγχουμε διάφορα κομμάτια εντολών ως προς το πότε και αν θα τρέξουν.
Μια συνθήκη μπορεί να είναι για παράδειγμα μια σύγκριση μεταξύ δύο μεταβλητών ή σύγκριση δύο αριθμών.

Υπάρχουν 3 τύποι συνθηκών:

* Συνθήκη <span style={{ color: '#d35763'}}> if </span>
* Συνθήκη <span style={{ color: '#d35763'}}> else </span>
* Συνθήκη <span style={{ color: '#d35763'}}> else if </span>

---

<h2 style="text-align: center;"> Συνθήκη <span style={{ color: '#d35763'}}> if </span> </h2>

Η συνθήκη if είναι η πιο απλή. Ελέγχει αν μια συνθήκη είναι αληθής.
Αν είναι, εκτελεί τις εντολές που έχουμε γράψει μέσα σε αυτό.

Παράδειγμα:

Στο παρακάτω παράδειγμα, θα εκτυπώσουμε ένα μήνυμα μόνο αν ο αριθμός είναι μεγαλύτερος από 5:

```cpp
#include <iostream>

using namespace std;

int main() {
    int x = 6;
    if (x > 5) {
        cout << "Ο αριθμός είναι μεγαλύτερος από 5";
    }
    return 0;
}
```

Ας αναλύσουμε τα στοιχεία της συνθήκης `if`.

<h3 style="text-align: left;"> Συνθήκη: `if (x > 5)` </h3>

Αυτό είναι το τμήμα της συνθήκης που ελέγχει αν ο αριθμός είναι μεγαλύτερος από 5.

<h3 style="text-align: left;"> Αγκύλες: `{ ... }` </h3>

Μέσα σε αυτές τις αγκύλες βάζουμε τις εντολές που θέλουμε να εκτελεστούν αν η συνθήκη είναι αληθής.
Οτιδήποτε βάλουμε εδώ θα εκτελεστεί μόνο αν η συνθήκη είναι αληθής
και οτιδήποτε βάλουμε εκτός αυτών των αγκυλών θα εκτελεστεί ανεξάρτητα από τη συνθήκη.

---

<h2 style="text-align: center;"> Συνθήκη <span style={{ color: '#d35763'}}> else </span> </h2>

Η συνθήκη else τρέχει όταν η συνθήκη if είναι ψευδής. Δεν είναι 
υποχρεωτική, αλλά μπορεί να χρησιμοποιηθεί για να εκτελέσει κώδικα
μόνο όταν η συνθήκη if ειναι ψευθής. Αν γράφαμε μία εντολή εκτός 
του else θα εκτελούνταν ανεξαρτήτως του τι θα γινόταν με το if.

Στο παρακάτω παράδειγμα, θα εκτυπώσουμε ένα μήνυμα αν ο αριθμός 
είναι μεγαλύτερος από 5,αλλά θα <br/> εκτυπώσουμε ένα διαφορετικό μήνυμα 
αν ο αριθμός είναι οτιδήπωτε διαφορετικό ( μικρότερος ή ίσος με 5):

```cpp
#include <iostream>

using namespace std;

int main() {
    int x = 3;
    if (x > 5) {
        cout << "Ο αριθμός είναι μεγαλύτερος από 5";
    } else {
        cout << "Ο αριθμός είναι μικρότερος ή ίσος με 5";
    }
    return 0;
}
```

---

<h2 style="text-align: center;"> Συνθήκη <span style={{ color: '#d35763'}}> else if </span> </h2>

Η συνθήκη <span style={{ color: '#d35763'}}> else if </span> βρίσκεται *πάντα* μεταξύ των συνθηκών <span style={{ color: '#d35763'}}> if </span> 
και <span style={{ color: '#d35763'}}> else </span>. Χρησιμοποιείται για να ελέγξει μια νέα συνθήκη αν η προηγούμενη συνθήκη ήταν ψευδής.
Μπορούμε να έχουμε όσες συνθήκες <span style={{ color: '#d35763'}}> else if </span> θέλουμε. Επιπλέον, πρέπει πάντα να είναι μια συνθήκη 
<span style={{ color: '#d35763'}}> else if </span> ακριβώς μετά μια συνθήκη <span style={{ color: '#d35763'}}> if </span> ή 
<span style={{ color: '#d35763'}}> else if </span> και πριν από μια συνθήκη else. Δεν μπορεί ούτε να "σταθεί" μόνη της αλλά ούτε και 
να μπει σε άλλη θέση μιάς και λειτουργεί σας ένα επιπλέον <span style={{ color: '#d35763'}}> if </span>.

Στο παρακάτω παράδειγμα, θα εκτυπώσουμε ένα μήνυμα αν ο αριθμός είναι μεγαλύτερος από 5,
αλλά θα εκτυπώσουμε ένα διαφορετικό μήνυμα αν ο αριθμός είναι μικρότερος ή ίσος με 5,
αλλά μεγαλύτερος από 0:

```cpp
#include <iostream>

using namespace std;

int main() {
    int x = -3;
    if (x > 5) {
        cout << "Ο αριθμός είναι μεγαλύτερος από 5";
    } else if (x > 0) {
        cout << "Ο αριθμός είναι μικρότερος ή ίσος με 5, αλλά μεγαλύτερος από 0";
    } else {
        cout << "Ο αριθμός είναι μικρότερος ή ίσος με 0";
    }
    return 0;
}
```
