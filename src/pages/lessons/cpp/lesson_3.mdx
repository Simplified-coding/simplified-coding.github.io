<h2 style="text-align: center;"> Τύποι Δεδομένων </h2>

Πριν δούμε τις μεταβλητές στη C++, θα πρέπει να δούμε τους τύπους δεδομένων που υπάρχουν.

Υπάρχουν 5 βασικοί τύποι δεδομένων στη C++:

| Τύπος Δεδομένων | Περιγραφή              |
|-----------------|------------------------|
| integer         | Ακέραιος αριθμός       |
| double          | Δεκαδικός αριθμός      |
| character       | Ένας χαρακτήρας        |
| string          | Κείμενο                |
| boolean         | Τιμή αληθούς ή ψεύδους |

Προσοχή!!! Υπάρχει διαφορά μεταξύ character και string. Με απλά λόγια τα strings είναι ακολουθίες (arrays) από characters. <br/>
Για την ώρα δεν υπάρχει λόγος να χρησιμοποιείτε characters διότι μπορείτε να κανετε ό,τι θέλετε και με strings.

<h2 style="text-align: center;"> Μεταβλητές </h2>

Οι μεταβλητές είναι ένα κομμάτι χώρου στη μνήμη, ο οποίος έχει έναν από τους 5 τύπους δεδομένων που είδαμε πριν. 
Οι μεταβλητές έχουν ένα όνομα, το οποίο πρέπει να είναι μοναδικό στο πρόγραμμα, και μπορούν να πάρουν μια τιμή
από τον τύπο δεδομένων που έχουν.

<h3 style="text-align: left;"> Ορισμός μεταβλητής </h3>

Για να ορίσουμε μια μεταβλητή, πρέπει να γράψουμε τον τύπο δεδομένων, το όνομα της μεταβλητής και την τιμή που θέλουμε
να πάρει η μεταβλητή. Αν δε θέλουμε να δώσουμε τιμή στη μεταβλητή, τότε απλά δε γράφουμε την τιμή.

Παράδειγμα:

```cpp
#include <iostream>

using namespace std;

int main() {
    int x = 5;
    double y = 3.14;
    char z = 'a';
    string w = "Hello World!";
    bool t = true;
    return 0;
}
```

<h3 style="text-align: left;"> Λειτουργίες μεταβλητών </h3>

Στο προηγούμενο μάθημα είχαμε μάθει διάφορες εργασίες με μαθηματικά. <br/>
Μπορούμε να εφαρμόσουμε ότι μάθαμε να κάνουμεμε το `cout` σε μεταβλητές.

Παράδειγμα:

```cpp
#include <iostream>
#include <cmath>

using namespace std;

int main() {
    int x = 5;
    int y = 3;
    int z = x + y;
    cout << z << endl;
    cout << x != y << endl;
    cout << x != y && x == y << endl;
    w = sqrt(x);
    cout << w << endl;
    return 0;
}
```

<h2 style="text-align: center;"> Εισαγωγή Δεδομένων (input) </h2>

Μέχρι τώρα, δίναμε εμείς τιμές στις μεταβλητές μας μέσα στο πρόγραμμα.
Τώρα όμως μπορούμε να δίνουμε <br/> τιμές στις μεταβλητές μας βάση του τι εισάγει ο χρήστης.
Για να το κάνουμε αυτό, χρησιμοποιούμε την εντολή `cin` με τον παράτω τρόπο:

```cpp
#include <iostream>

using namespace std;

int main() {
    int x;
    cout << "Εισάγετε έναν αριθμό: ";
    cin >> x;
    cout << "Εισάγατε: " << x << endl;
    return 0;
}
```
