<h2 style="text-align: center;"> Τι είναι οι φόρμες στην HTML? </h2>

Με το όρο "φόρμες" εμείς εννωούμε όλα τα πεδία και αντικείμενα που μπορείς να χρεισημοποιήσιες σε 
ένα ερωτηματολόγιο, Log-In σελίδα ή απλώς σε μία σελίδα για αξιολόγιση. Σε αντίθεση με πολλά από 
τα προηγούμενα μαθήματα δεν είναι κάτι που θα χρεισιμοποιείς σε κάθε ιστοσελίδα (ή και όχι, από 
εσένα εξαρτάται)και δεν ανίκουν στα βασικά στοιχεία. Αυτο το αναφέρω διότι θα είναι αρκετά. Αν 
έχεις χρόνο και διάθεση θα ήταν ιδανικό να τα δεις όλα για να έχεις μια ιδέα των δυνατοτήτων της 
HTML αλλά μην ανυσηχείς αν δειςς μόνο εκείνα που θα ήθελες να φτιάξεις. 

---

<h2 style="text-align: center;"> Πεδίο εισαγωγής κειμένου </h2>

Όπως φανερώνει και ο τίτλος είναι το πιο γνωστό πεδίο. Σε αυτό οχρήστης μπορεί να γράψει το δικό του κείμενο. Ειναι χρήσιμο για
ερωτήσεις ανοιχτού τύπου σχεδόν παντού από ερωτηματολόγια μέχρι και για σελίδες για κρητική.

Για να δημιουργήσουμε αυτό το πεδίο χρησιμοποιούμς το ```textarea``` tag:

```html
<textarea> </textarea>
```

<textarea> </textarea>
Έτσι θα εμφανιζόταν σε μία ιστοσελίδα να γράφαμε μόνο τα tags. Όπως βλέπεις είναι αρκετά άσχημο και δύσχρηστο για κάπιον 
να το χρησιμοποιήσει όπως ειναι φτιαγμένο αυτόματα (άσπρο υπόβαρθο με άσπρο κείμενο) οπότε πρέπει να το επεργαστούμε μέσω της CSS. 
Όμως επειδή αυτό ειναι υλικό για την σειρά της CSS (Δείτε το αντιστοιχο μάθημα) για την ώρα θα δούμε τι αλλο μπορούμε να κάνουμε με αυτο το πεδίο.
<br></br>
<h3> placeholder attribute </h3>

Για αρχή μπορείς να του ορίσεις να εμφανίζει ένα κείμενο το οποίο θα εξαφανίζεται όταν ο χρήστης γράφει κάτι, με σκοπό να του 
δίνεις να καταλάβει τι θα πρέπει να γράψει σε αυτό το πεδίο. Ίσως να μην κατάλαβες τι εννοώ (το καταλαβαίνω, δεν ειμαι και ο 
καλύτερος στις περιγραφές), γι' αυτό ας δούμε ένα παράδειγμα του πως θα μοιάζει κάτι τέτοιο:

<textarea placeholder="Πώς σας φαίνεται η ιστοσελίδα μας?" style="background-color: #54d683; color: #4b2366"> </textarea>

<p> Αυτό μπορούμε να το φτιάξουμε χρησιμοποιόντας ένα νέο attribute, το ```placeholder``` attribute. Τώρα δεν έχουν μείνει 
πολλά να εξηγήσω, απλως εκτυπώνει ό,τι βάζεις ανάμεσα στα εισαγωγικά (αυτό λέγεται value του attribute).
Να πως ακριβώς το δημιουργούμε στην HTML: </p>


```html
<textarea placeholder="Πώς σας φαίνεται η ιστοσελίδα μας?"> </textarea>
```

<br></br>
<h3> rows attribute </h3>

<p> Αυτο είναι ακόμα πιο απλό. Αυτό που κάνει το rows attribute ειναι το να ορίζει ως προεπιλογή το πόσες σειρές θέλεις
να έχει το πεδίο σου. Φυσικά ο χρήστης μπορεί να το αλλάξει αν θέλει αλλά έτσι κάνει το κέιμενο να χωραέι καλύτερα 
στο πλαίσιο με απολέσμα να φαίνεται και καλύτερα. </p>

Και κάπως έτσι το δημιουργούμε:

```html
<textarea placeholder="Πώς σας φαίνεται η ιστοσελίδα μας?" rows=" value "> </textarea>
```

<textarea placeholder="Πώς σας φαίνεται η ιστοσελίδα μας?" rows="5" style="background-color: #54d683; color: #4b2366"   > </textarea>

<h2> Πεδίο πολλαπλής επιλογής (Checkboxes) </h2>

Τα checkboxes είναι μία άλλη ιστορία, αρκετά διαφορετικά. Για αρχή δεν έχουν δικό τους tag αλλα χρησιμοποιούν ένα κοινό tag για όλα
τα πεδία με επιλογή. Εδώ για να ξεχωρίσει το τι είδους πεδου ειναι η HTML πρέπει να χρησιμοποιείσουμε ένα νέο attribute, το ```type```. 
Στην συγκεκριμένη περίπτωση (για τα checkboxes) θα ορίσουμε ως value του attribute την λέξη `checkbox` (ξέρω... πρωτότυπο) και μετά 
το input tag το κείμενο το οποίο θέλουμε να υπάρχει δίπλα στο checkbox. 

Έχει περάσει ώρα χωρίς ένα παράδειγμα οπότε μιάς και τα ειπα όλα μαζεμένα ορίστε το τι πρέπει να γράψεις 
για να εμφανιστεί όπως και  πως θα μοιάζει:

```html
<input type="checkbox"> Καρότα
<input type="checkbox"> Πιπεριές
<input type="checkbox"> Ντομάτες
<input type="checkbox"> Μαρούλι
```

<input type="checkbox"> Καρότα </input>
<input type="checkbox"> Πιπεριές </input>
<input type="checkbox"> Ντομάτες </input>
<input type="checkbox"> Μαρούλι </input>

<h2> Πεδίο μοναδικής επιλογής (Radios) </h2>

Αυτά τα πεδία είναι παρόμοια με τα checkboxes με την μοναδική διαφορά ότι δεν μπορείς να επιλέξεις πολλά πεδία
αλλά μόνο ένα. Αυτο είναι καλύτερο για μία φόρμα ή για ένα ερωτηματολόγιο στο οποίο επιλέγεις την καταλληλότερη απάντηση.

Δημιουργούνται με τον ίδιο ακριβώς τρόπο με τα chackboxes με την διαφορά ότι τώρα στο value 
του ```type``` attribute θα γράψουμε ```radio``` αλλά επίσης χρειάζεται και ένα ακόμα attribute, το ```name```. Αυτο υπάρχει
για μας <s> δυσκολέυει την ζωη </s> επιτρέπει να έχουμε πολλά radios στην ίδια σελίδα χωρίς να περδεύονται μεταξύ τους.
Xάρη σε αυτό το attribute τα radios με το ίδιο name value μόνο ένα από αυτά μπορεί να είναι checked.
 Δείτε το ίδιο παράδειγμα με πριν αλλά σε ως πεδίο επιλογής radio:

```html
<input type="radio" name="flexRadioDefault"> Καρότα
<input type="radio" name="flexRadioDefault" checked> Πιπεριές
<input type="radio" name="flexRadioDefault"> Ντομάτες
<input type="radio" name="flexRadioDefault"> Μαρούλι
```

<input type="radio" name="flexRadioDefault"> Καρότα</input>
<input type="radio" name="flexRadioDefault" checked> Πιπεριές</input>
<input type="radio" name="flexRadioDefault"> Ντομάτες</input>
<input type="radio" name="flexRadioDefault"> Μαρούλι</input>

<h2> Πεδίο εύρους (Range input) </h2>

Με αυτό το πεδίο μπορείς να επιτρέπεις στους χρήστες να επιλέξουν μεταξύ τιμών ενός εύρους τιμών. Αυτό μπορεί να 
είναι πολύ χρησιμο σε αιτήσεις, για παράδειγμα, για να εισάγει το χρήστης την ηλικία του ή το έτος γεννήσεως. 

Η δημιουργία ενός range input δεν διαφέρει και πολύ από τα checkboxes με την μόνη διαφορά ότι ορίζουμε το ```type``` value σε ```range```.
Τώρα ας δούμε όπως και στα προηγούμετα inputs το πως το φτιάχνουμε όπως και το πως μοιάζει:

```html
<input type="range"> Τιμές (0-100) 
```
<input type="range"> Τιμές (0-100) </input>

Φυσικά μπορείς να βάλεις τιμές στην αρχή και στο τέλος της μπάρας αλλά αυτό δεν είναι υλικό για 
το μάθημά μας, ήδη ειναι μεγάλο, δεν χρειάζεαι να το κάνω χειρότερο.

<h2> πεδίο email - password </h2>

Αυτό τα 2 πεδία ειναι τόσο παρόμοια, για αυτό θα τα αναφέρω μαζί. θα εκπλαγείς αλλά και αυτό είναι ίδιο με τα checkboxes απλώς ορίζεις
το ```type``` value σε ``` email```. Ειναι προφανές το πότε μας είναι χρήσιμο. Ακόμα, όπως και στο πεδίο κειμένου μπρούμε να χρισημοποιήσουμε
το placeholder attribute για να δώσουμε ένα παράδειγμα του τι προορίζεται για αυτό το πεδίο.
Ορίστε το παράδειγμα μαζί με το πώς εμφανίζεται:

```html
<input type="email" placeholder="name@example.com">
```

<input type="email" placeholder="name@example.com" style="background-color: #54d683; color: #4b2366"></input>


Αντίστοιχα στο password γράφουμε ακριβώς το ίδιο αλλά με το ```type``` value να ορίζεται σε ```password```. Η μόνη διαφορά με το πεδίο 
του email ειναι ότι εδώ αυτά που γράφεις δεν εμφανλίζονται αλλά σητν θέση τους εμφανιζονται τελίτσες. Ορίστε το δύδιμο (παράδειγμα - πως εμφανίζονται)

```html
<input type="password">
```

<input type="password" style="background-color: #54d683; color: #4b2366"></input>
