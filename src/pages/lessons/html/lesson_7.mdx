<h2 style="text-align: center;"> Οργανωμένες λίστες </h2>

Με τις λίστες της html Μπορούμε να φτιάξουμε....λίστες!! Δεν έχει να πω και πολλά στην εισαγωγή. 
Ας ξεκινήσουμε με ένα απλό παράδειγμα:
```html
<ol>
  <li>Όνομα</li>
  <li>Διεύθυνση</li>
  <li>Αριθμός τηλεφώνου</li>
</ol>
```
Όπως βλέπεις αρχίζουμε και κλείνουμε την λίστα μας με τα tags της οργανωμένης λίστας ```<ol>``` (από τα αρχικά της οργανωμένης λίστας 
στα αγγλικά, ordered list). Μέσα σε αυτα τα tags, κάθε ένα στοιχείο που θέλουμε να τυπώσουμε πρέπει να το βάλουμε μέσα σε ειδικά tags 
για να καταλάβει ο browser οτι θα αλλάξει σειρά στοιχείο. Αυτά τα tags είναι τα list item tags: ```<li></li>```. <br/>
Το παραπάνω παράδειγμα σε μία ιστοσελίδα θα τυπονώταν κάπως έτσι:
<ol>
  <li>Όνομα</li>
  <li>Διεύθυνση</li>
  <li>Αριθμός τηλεφώνου</li>
</ol>

<h2 style="text-align: center;"> Ανοργάνωτες λίστες </h2>

Οι ανοργάνωτες λίστες είναι παρόμοιες με τις οργανωμένες με την διαφορά ότι αντί για αρίθμηση, τα στοιχεία χωρίζονται με τελίτσες (bullet points).



Ακόμα αν έχουμε μια ανοργάνωτη λίστα μέσα σε μία άλλη ανοργάνωτη λίστα τότε οι τελιτσες μπροστά από τα στοιχεία θα αλλάξει αυτόματα ώστε
να μπορούμε να ξεχωρίζουμε του σε ποια λίστα ανήκει κάθε στοιχείο (nesting lists). Ας δούμε και ένα παράδειγμα από μία τέτοια περίπτωση:
Ας δούμε ένα παράδειγμα μίας ανοργάνωτης λίστας:
```html
<ul>
  <li>Πλήρωσε τους λογαριασμούς</li>
  <li>Πήγαινε στο super-market</li>
    <ul>
        <li>Αγόρασε ντομάτες</li>
        <li>Αγόρασε ψωμί</li>
        <li>Αγόρασε τυρί</li>
        <ul>
            <li>Gouda</li>
            <li>Κεφαλοτύρι</li>
        </ul>
        <li>Αγόρασε καρότα</li>
    </ul>
  <li>Πλείνε στο αμάξι</li>
</ul>
</ul>
```
Το παραπάνω παράδειγμα θα τύπωνε στην ιστοσελίδα αυτο:
<ul>
  <li>Πλήρωσε τους λογαριασμούς</li>
  <li>Πήγαινε στο super-market</li>
    <ul>
        <li>Αγόρασε ντομάτες</li>
        <li>Αγόρασε ψωμί</li>
        <li>Αγόρασε τυρί</li>
        <ul>
            <li>Gouda</li>
            <li>Κεφαλοτύρι</li>
        </ul>
        <li>Αγόρασε καρότα</li>
    </ul>
  <li>Πλείνε στο αμάξι</li>
</ul>

<h2 style="text-align: center;"> Πίνακες </h2>

Οι πίνακες είναι ένα πολύ χρήσιμο και εύχρηστο εργαλίο της HTML. Με αυτό μπορούμε να οργανώσουμε δεδομένα και να τα τυπώσουμε στην ιστοσελίδα μας
με έναν ευπαρουσίαστο και όμορφο τρόπο. Ακόμα είναι πολύ εύκολη η τρόποποίησή τους διότι χωρίζονται σε αρκετά μέρη από μόνοι τους οπότε κάθε κελί,
σειρά, τίτλος και στήλη μπορεί να επεξεργαστεί ξεχωριστά. <br/>
Ορίστε ένα παράδειγμα ενός πίνακα:
```html
<table>
  <th>
    <td>Κατερίνα</td>
    <td>Γιώργος</td>
    <td>Στράτος</td>
    <td>Μαρία</td>
  </th>
  <tr>
    <td>Αγαπημένο χρώμα</td>
    <td>Πορτοκαλί</td>
    <td>Κίτρινο</td>
    <td>Μοβ</td>
    <td>Πράσινο</td>
  </tr>
  <tr>
    <td>Age</td>
    <td>16</td>
    <td>9</td>
    <td>10</td>
    <td>5</td>
  </tr>
  <tr>
    <td>Τόπος κατοικίας</td>
    <td>Θεσσαλονίκη</td>
    <td>Αθήνα</td>
    <td>Πάτρα</td>
    <td>Χίο</td>
  </tr>
  <tr>
    <td>Αγαπημένο φαγητό</td>
    <td>Παστίτσιο</td>
    <td>Κεφτεδάκια</td>
    <td>Σουβλάκι</td>
    <td>Μακαρονάδα</td>
  </tr>
</table>
```
Το ξέρω, είναι μεγάλο αλλά σιγά-σιγά θα το εξηγήσουμε. Όπως βλέπεις έχει μία μορφή πολύ παρόμοια με μία λίστα. Αυτό γιατί έχει ίδια λειτουργεία
Δηλάδη ο browser διαβάζει τα πάντα από έξω προς τα μέσα. Αρχικά ας ξεκινήσουμε με τα εύκολα. Όλος ο πίνακας μπάινει μέσα σε ένα ```<table>``` tag.
<br/>
Στην συνέχεια (προς τα μέσα) βλέπουμε το ```<th>``` tag (από τον αγγλικό όρο table head) με το οποίο δηλώνουμε ότι ό,τι γράψουμε εκεί μέσα θα 
είναι τίτλος αυτού του πίνακα στην πρώτη σειρά του πίνακα. 
<br/>
Αν βουτύξουμε ακόμα βαθύτερα σε αυτόν τον πίνακε θα δούμε μερικά ```td``` tags (από τον αγγλικό όρο table data). Αυτό το tag έχει ακριβώς 
την ίδια λειτουργεία με το ```<li>``` tag από τις λίστες, δηλώνει στον browser ότι αλλάζουμε στοιχείο. Αυτό μπαίνει μέσα και στα ```<th>``` 
tags αλλά και στα ```<tr>``` tags. 
<br/>
Μιάς και τα αναφέραμε ας δούμε τι είναι αυτά τα ```<tr>``` tags (από τον αγγλικό όρο table row). Αυτά δηλώνουν ότι δεν βρησκόμασε πια στον τίτλος
του πίνακα αλλά αλλάζουμε σειρά και προχωράμε στο κύριο μέρος του πίνακα. Κάθε φορά που το γράφουμε θα άλλαζει η σειρά και τα στοιχεία μέσα στα
```<td>``` tags θα γεμίζουν τις στήλες από αριστερά προς τα δεξιά. Έτσι αν αγήσουμε κάποιο κενό στοιχείο απλώς θα παραλείψει εκείνη στην σειρά.
