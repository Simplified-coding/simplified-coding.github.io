<h1 style="text-align: center;"> Μάθημα 2 </h1>

<h2 style="text-align: center;"> Η κεφαλή της HTML </h2>

---

<h2 style="text-align: center;"> Τι είναι η κεφαλή; </h2>

Καλωσήρθες σε άλλο ένα μάθημα HTML!
Μπορεί να διάβασες τον τίτλο και να αναρωτήθηκες τι είναι η κεφαλή.
Μπορώ να σε βεβαιώσω ότι δε θα κάνουμε βιολογία.
Πάμε λοιπόν πίσω στο παράδειγμα του προηγούμενου μαθήματος!

```html
<!DOCTYPE html>
<html>
<head>
    <title>Ετήσιο Φεστιβάλ Τεχνολογίας 2024</title>
    <link rel="stylesheet" href="style.css">
</head>

<body>
    <main class="event-invitation">
        <h1>Καλώς ήρθατε στο Φεστιβάλ Τεχνολογίας</h1>
        <p class="date-time">5 Μαρτίου, 2023, στις 10:00 ΜΜ</p>
        <p class="venue">Κέντρο τεχνολογικής καινοτομίας, Silicon Valley</p>
        <p class="description">
            Έλα μαζί μας για μία μέρα γεμάτη με επίκαιρες συζητήσεις, ευκαιρίες δικτύωσης, και τα τελευταία νέα επάνω στην τεχνολογία.
        </p>
        <button class="rsvp-button">Δήλωσε συμμετοχή</button>
    </main>
</body>
</html>
```

Όπως βλέπεις, υπάρχει ένα tag που γράφει `<head>`.
Head στα αγγλικά σημαίνει κεφάλι.
Οπότε λοιπόν και η HTML έχει ένα κεφάλι!
Τι υπάρχει μέσα στο κεφάλι αυτό; Γιατί το έχουμε;
Πάμε να δούμε!

Μέσα στην κεφαλή βάζουμε κώδικα που δε θέλουμε η ιστοσελίδα μας να γράφει.
Τι θα θέλαμε να μη γράφει η ιστοσελίδα μας;
Η κεφαλή περιέχει πληροφορίες για την ιστοσελίδα μας.
Σκέψου το σαν τον εγκέφαλο σου.
Δεν τον βλέπουν οι άλλοι, αλλά έχει πληροφορίες μέσα για εσένα.
Κάπως έτσι είναι και το head.
Πάμε να δούμε τι μπορούμε να βάλουμε στο κεφάλι της σελίδας μας.

---

<h2 style="text-align: center;"> Τίτλος σελίδας </h2>

Έχεις παρατηρήσει ποτέ ότι όταν μπαίνεις σε μια ιστοσελίδα,
στην καρτέλα του προγράμματος περιήγησης σου εμφανίζεται ένας τίτλος;
Για παράδειγμα, η σελίδα αυτή έχει τον τίτλο Simplified Coding.

Να μια εικόνα για να καταλάβεις:

![Τίτλος σελίδας](https://cloud-k5q4o2868-hack-club-bot.vercel.app/0image.png)

Πως το βάζουμε λοιπόν αυτό;
Με το ```<title>``` tag.
Στο κεφάλι της σελίδας φυσικά!
Αλλά για να βάλουμε το title πρέπει πρώτα να συναρμολογήσουμε το κεφάλι.
Αυτό θα γίνει με το tag ```<head>```.
Και που θα μπεί το title;
Μέσα στο head!
Αν θυμάσαι από το προηγούμενο μάθημα,
στην ουσία το title θα είναι περιεχόμενο του head.

Ας δούμε και ένα παράδειγμα:

```html
<!DOCTYPE html>

<head>
    <title>Simplified Coding</title>
</head>
```

Και εμείς έτσι έχουμε βάλει τον τίτλο στην ιστοσελίδα μας!


---

<h2 style="text-align: center;"> Meta </h2>

Ήρθε η ώρα να μπούμε στα βαθιά.
Το meta (που δεν έχει καμία σχέση με τη γνωστή εταιρία) σημαίνει metadata.
Να πω την αλήθεια δε βρήκα κάποια καλή μετάφραση, οπότε ας τα λέμε απλά metadata.
Αυτά λοιπόν είναι δεδομένα σχετικά με την ιστοσελίδα μας.
Δεδομένα όπως την περιγραφή της σελίδας,
λέξεις κλειδιά που την περιγράφουν, τη γλώσσα της και πολλά ακόμα.
Εμείς θα δούμε μερικά, όμως μπορείς εσύ να βρείς και άλλα στο ![W3Schools](https://www.w3schools.com/tags/tag_meta.asp)
(είναι στα αγγλικά).

Πώς φτιάχνουμε λοιπόν ένα meta;
Με το tag ```<meta>```.
Αλλά πριν είπαμε ότι το meta μπορεί να πάρει πολλά και διαφορετικά είδους δεδομένα.
Πώς θα του πούμε τι είδους δεδομένα θέλουμε να πάρει;
Αυτό θα γίνει με τα attributes, που είχαμε δει στο προηγούμενο μάθημα.
Ας δούμε αρχικά πως βάζουμε το meta.

```html
<!DOCTYPE html>

<head>
    <title>Simplified Coding</title>
    <meta>
</head>
```

Τώρα, ας αναλύσουμε τι μπορούμε να βάλουμε στο meta.

---

<h3 style="text-align: center;"> Περιγραφή </h3>

Το πιο σημαντικό πράγμα που μπορούμε να βάλουμε στο meta είναι η περιγραφή.
Είναι σημαντικό γιατί με μια καλή περιγραφή,
η ιστοσελίδα σου μπορεί να εμφανιστεί πιο πάνω στα αποτελέσματα της αναζήτησης σε μηχανές αναζήτησης (πχ Google).

Ας δούμε πως μπορούμε να βάλουμε μια περιγραφή στο meta.

```html
<!DOCTYPE html>

<head>
    <title>Simplified Coding</title>
    <meta name="description" content="Μάθε προγραμματισμό, δωρεάν στα ελληνικά">
</head>
```

Εδώ πρέπει να έχουμε 2 attributes.
Το πρώτο είναι το name, που λέει τι είδους δεδομένο θα πάρει το meta.
Στην περίπτωσή μας, θέλουμε να πάρει την περιγραφή της ιστοσελίδας.
Το δεύτερο είναι το content, που λέει το περιεχόμενο του meta.
Στην περίπτωσή μας, είναι το τι θα γράφει η περιγραφή.

---

<h3 style="text-align: center;"> Λέξεις κλειδιά </h3>

Κάτι άλλο που μπορούμε να βάλουμε στο meta είναι οι λέξεις κλειδιά.
Αυτές είναι λέξεις που περιγράφουν την ιστοσελίδα μας.
Με αυτές τις λέξεις, η ιστοσελίδα μας μπορεί να εμφανιστεί πιο πάνω στα αποτελέσματα της αναζήτησης.
Λέξεις κλειδιά μπορεί να είναι πολλές, αλλά πρέπει να είναι σχετικές με το περιεχόμενο της ιστοσελίδας.

Ας δούμε πως μπορούμε να βάλουμε λέξεις κλειδιά στο meta.

```html
<!DOCTYPE html>

<head>
    <title>Simplified Coding</title>
    <meta name="description" content="Μάθε προγραμματισμό, δωρεάν στα ελληνικά">
    <meta name="keywords" content="προγραμματισμός, δωρεάν, ελληνικά, html, μαθήματα">
</head>
```

Όπως βλέπεις, χωρίζουμε τις λέξεις κλειδιά με κόμμα.

---

<h3 style="text-align: center;"> Γλώσσα </h3>

Το τελευταίο που θα δούμε είναι η γλώσσα.
Αυτό είναι σημαντικό γιατί με την κατάλληλη γλώσσα,
η ιστοσελίδα μας μπορεί να εμφανιστεί σε άτομα που μιλάνε αυτή τη γλώσσα.

Ας δούμε πως μπορούμε να βάλουμε τη γλώσσα στο meta.

```html
<!DOCTYPE html>

<head>
    <title>Simplified Coding</title>
    <meta name="description" content="Μάθε προγραμματισμό, δωρεάν στα ελληνικά">
    <meta name="keywords" content="προγραμματισμός, δωρεάν, ελληνικά, html, μαθήματα">
    <meta name="language" content="el">
</head>
```

Τα ελληνικά τα συμβολίζουμε με το el.
Μπορείς να βρείς το κωδικό γλώσσας που θέλεις
![στο W3School με βάση την πιστοποίηση ISO 639-1](https://www.w3schools.com/tags/ref_language_codes.asp).
