<h2 style="text-align: center;"> Dividers </h2>

Όταν φτιάχνουμε μία ιστοσελίδα πρέπει να την χωρίσουμε σε διάφορα μέρη για να μπορούμε να τα επεξεργαστούμε 
ευκολότερα στην CSS, να οργανόσουμε τον κώδικα σε κομμάτια
και να είναι ευκολότερο για εμάς να θυμόμαστε το που βρήσκεται ένα μέρος του κώδικα που 
ψάχνουμε. Πέρα από τα: head, body και footer (θα το δούμε αυτό μετά)• η HTML έχει και τo ```div``` με το οποίο μπορούμε 
να χωρίσουμε οποιοδήποτε κομμάτι του κώδικά μας. <br/>Ορίστε ένα παράδειγμα για να γίνει πιο κατανοητό:

```html
<p>Στο ηλιακό μας σύστημα υπάρχουν πάνω από 200 
δορυφόροι. Η Γη έχει 1 δορυφόρο, την Σελήνη.</p>

<div id="earth">
  <h2>Η Γη</h2>
  <p>Μία σφαίρα με μπόλικο μπλε που ζούμε εμείς.</p>
  <p>Εδώ ζουν πάνω από 8 δισσεκατομμύρια άνθρωποι.</p>
</div>

<div id="moon">
    <h2>H Σελήνη</h2>
    <p>Η Σελήνη είναι το μόνο ουράνιο σώμα που έχει 
        πατήσει το πόδι του ο άνθρωπος.</p>
</div>
```

Στο παραπάνω παράδειγμα χωρίζουμε το κείμενό μας σε 3 κομμάτια (αυτό χωρις div και σε αλλα 2 με divs). 
Έτσι μπορούμε για παράδειγμα να κάνουμε τα γράμματα στο κομμάτι που μειλάμε για την Γη μπλε ενώ το υπόλοιπο κείμενο να μίνει ανεπιρέαστο.

---

<h2 style="text-align: center;"> Id tag</h2>

Μπορεί να μην είμουν απολύτως ειληκρινής πριν. Δεν μπορείς να επεξεργαστείς ξεχωριστά τα divs χωρίς την βοήθεια ενός id tag. 
Αν πρόσεξες στο προηγούμενο παράδειγμα είχα κρυμμένα μερικά id tags με τα οποία ξεχωρίζουμε τo div για την Γη με το div για Σελήνη.
Συνήθως όταν δημιουργούμε ένα div θεωρούμε σχεδόν βέβαιο ότι θα χρειαστεί να το συνοδεύσουμε <s>με λίγο κρασί</s> με ένα id. Στα ids 
ακόμα και αν φτιάχνουμε μία ιστοσελίδα στα ελληνικά, να χρησιμοποιούμε λατινικούς χαρακτήρες για να ειναι κατανοητά από την css. Τέλος,
δεν μπορείς να έχεις πάνω από 1 id σε ένα div/element, κάθε id είναι μαναδικό σε κάθε ιστοσελίδα.

---

<h2 style="text-align: center;"> class tag</h2>

Τα classes δεν διαφέρουν σχεδόν καθόλου με τα ids. Αρχικά σε αντίθεση με τα IDs, μπορείς να έχεις πάνω από 1 class σε ένα div/tag έτσι 
μέσω της CSS κάποια elements θα έχουν μια ιδότητα (πχ να είναι κόκκινα τα γράμματα) και κάποια άλλα 2 ids να έχουν και αυτή την ιδιότητα 
αλλά και μία άλλη μοναδική (πχ και κόκκινα γράμματα αλλά και κεντραρισμένα), μπορείς; να πάρετε μία ιδέα από το παρακάτω παράδειγμα. Επίσης 
τα classes είναι πιο εύχρηστα όταν θέλουμε ένα αντικείμενο να το επεξεργαστούμε μέσω της javascript. Τέλος, μια ακόμα διαφορά είναι ότι ενώ 
τα ids μπαίνουν σχεδόν αποκληστικά σε divs, τα classes μπορούν να μπουν σχεδόν σε ολα τα elements (```<p>``` ```<h>``` ```<div>``` ```<head>```).

```html
<p>Στο ηλιακό μας σύστημα υπάρχουν πάνω από 200 
δορυφόροι. Η Γη έχει 1 δορυφόρο, την Σελήνη.</p>

<div class="celestial-body, palnet">
  <h2>Η ΓΗ</h2>
  <p>Μία σφαίρα με μπόλικο μπλε που ζούμε εμείς.</p>
  <p>Εδώ ζουν πάνω από 8 δισσεκατομμύρια άνθρωποι.</p>
</div>

<div class="celestial-body">
    <h2>H Σελήνη</h2>
    <p>Η Σελήνη είναι το μόνο ουράνιο σώμα που έχει 
        πατήσει το πόδι του ο άνθρωπος.</p>
</div>
```
