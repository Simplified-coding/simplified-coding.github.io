<h2 style="text-align: center;"> ```<body>``` tag </h2>

Καλωσήρθες σε άλλο ένα μάθημα HTML!
Στο προηγούμενο μάθημα είδαμε τα ```head``` tags.
Τώρα θα επικεντρωθούμε στα tags που βρήσκονται μέσα στο ```body``` tag ( ```p``` και ```h``` tags ).

```html
<!DOCTYPE html>
<html>
<head>
    <title>Ετήσιο Φεστιβάλ Τεχνολογίας 2024</title>
    <link rel="stylesheet" href="style.css">
</head>

<body>
    <main class="event-invitation">
        <h1>Καλώς ήρθατε στο Φεστιβάλ Τεχνολογίας</h1>
        <p class="date-time">5 Μαρτίου, 2023, στις 10:00 ΜΜ</p>
        <p class="venue">Κέντρο τεχνολογικής καινοτομίας, Silicon Valley</p>
        <p class="description">
            Έλα μαζί μας για μία μέρα γεμάτη με επίκαιρες συζητήσεις, ευκαιρίες 
            δικτύωσης, και τα τελευταία νέα επάνω στην τεχνολογία.
        </p>
        <button class="rsvp-button">Δήλωσε συμμετοχή</button>
    </main>
</body>
</html>
```

Όπως βλέπεις και μόνος σου στο ```body``` βρήσκεται το κομμάτι της ιστοσελίδας που βλέπει ο χρήστης
Τα 2 βασικά στοιχεία που χρησιμοποιούμε σε αυτό πιο συχνά είναι το ```p``` και το ```h``` tag τα οποία θα δούμε αμέσως τώρα.

---

<h2 style="text-align: center;"> ```<h>``` tag (Κεφαλίδα/Τίτλος) </h2>

Όταν γράφεις ένα άρθρο ή ένα blog δεν πρέπει ο αναγνώστης να ξέρει το θέμα της σελίδας;
Αυτό στην html γίνεται με τα headings. Με αυτά φτιάχνουμε τους τίτλους στην HTML. 
Η HTML διαθέτει 6 δειφορετικά μεγέθη γράμματων τα οποία είναι:

<h1>h1 (32 px)</h1>
<h2>h2 (24 px)</h2>
<h3>h3 (18.72 px)</h3>
<h4>h4 (16 px)</h4>
<h5>h5 (13.28 px)</h5>
<h6>h6 (10.72 px)</h6>

Φυσικά όλα τα μεγέθη των headers μπορούν να επεξεργαστούν με την CSS.
Μπορείς μετά τα μαθήματα της HTML, να δείς <a href="https://simplifiedcoding.org/lessons/view/css" target="_blank"> τα μαθήματά μας στην CSS </a> για να μάθεις περισσότερα.
Έτσι ώστε να έχεις στην ιστοσελίδα σου τα δικά σου μεγέθη τίτλων.

---

<h2 style="text-align: center;"> ```<p>``` tag (paragraph) </h2>

Με το ```<p>``` element, χωρίζουμε παραγράφους. Φυσικά και μπορούμε να τα χωρίσουμε με άλλους τρόπους 
αλλά χρησιμοποιόντας το ```<p>``` element ο browser αυτόματα μας αγφήνει ένα κενό πριν και μετά από την παράγραφο.
```html
<p>Αυτη είναι μία παράγραφος που μιλάει για την 
    κλιματική αλλάγή.</p>
<p>Αυτή ειναι μια άλλλη παράγραφος η οποία 
    μιλάει για το λιώσιμο ων πάγων </p>
```

<p>Αυτη είναι μία παράγραφος που μιλάει για την 
    κλιματική αλλάγή.</p>
<p>Αυτή ειναι μια άλλλη παράγραφος η οποία 
    μιλάει για το λιώσιμο ων πάγων </p>
<br/>
Αν παρατηρήσεται και μαγαλώσετε ή μικρίνεται το παράθυρο του μαθήματος θα δείτε ότι ο browser αυτόματα γεμίζει 
όσο χώρο έχει σε μία σειρά και αφήνει το κενό, για το οποίο μιλούσαμε, πριν και μετά της παραγράφου που φράψαμε.
