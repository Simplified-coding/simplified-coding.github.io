<h2 style="text-align: center;"> ```<h>``` element (Κεφαλίδα/Τίτλος) </h2>

Όταν γράφεις ένα άρθρο ή ένα blog δεν πρέπει ο αναγνώστης να ξέρει το θέμα της σελίδας;
Αυτό στην html γίνεται με τα heads. Με αυτά φτιάχνουμε τους τίτλους στην html. 
Η html διαθέτει 6 δειφορετικά μεγέθη γράμματων τα οποία είναι:
<h1>h1 (32 px)</h1>
<h2>h2 (24 px)</h2>
<h3>h3 (18.72 px)</h3>
<h4>h4 (16 px)</h4>
<h5>h5 (13.28 px)</h5>
<h6>h6 (10.72 px)</h6>

Φυσικά όλα τα μεγέθη των headings μπορούν να επεξεργαστούν από την css έτσι ώστε να έχεις στην ιτοσελίδα σου τα δικά σου μεγέθη τίτλων για να διευκολυνθείς.

---

<h2 style="text-align: center;"> ```<p>``` element (Παράγραφος) </h2>

Με το ```<p>``` element, όπως είναι προφανές, χωρίζουμε παραγράφους. Φυσικά και μπορούμε να τα χωρίσουμε με άλλους τρόπους 
αλλά χρησιμοποιόντας το ```<p>``` element ο browser αυτόματα μας αγφήνει ένα κενό πριν και μετά από την παράγραφο.
```html
<p>Αυτη είναι μία παράγραφος που μιλάει για την 
    κλιματική αλλάγή.</p>
<p>Αυτή ειναι μια άλλλη παράγραφος η οποία 
    μιλάει για το λιώσιμο ων πάγων </p>
```

<p>Αυτη είναι μία παράγραφος που μιλάει για την 
    κλιματική αλλάγή.</p>
<p>Αυτή ειναι μια άλλλη παράγραφος η οποία 
    μιλάει για το λιώσιμο ων πάγων </p>
<br/>
Αν παρατηρήσεται και μαγαλώσετε ή μικρίνεται το παράθυρο του μαθήματος θα δείτε ότι ο browser αυτόματα γεμίζει 
όσο χώρο έχει σε μία σειρά και αφήνει το κενό, για το οποίο μιλούσαμε, πριν και μετά της παραγράφου που φράψαμε.
