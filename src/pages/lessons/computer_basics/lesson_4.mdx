<h2 style="text-align: center;"> Δεδομένα Αρχείων και Καταλόγων </h2>

Όταν αποθηκεύετε δεδομένα σε έναν υπολογιστή, αυτά τα δεδομένα οργανώνονται σε μονάδες που ονομάζονται αρχεία και κατάλογοι (ή φάκελοι). Ας δούμε πώς λειτουργούν:

<h3 style="text-align: left;"> Αρχεία </h3>
Ένα αρχείο είναι μια μονάδα που περιέχει πληροφορίες
όπως κείμενο, εικόνες, μουσική ή βίντεο.
Αντίστοιχα, ένα αρχείο μπορεί να είναι ένα έγγραφο Word, μια φωτογραφία JPEG, ένα τραγούδι MP3 κ.λπ.

<h3 style="text-align: left;"> Κατάλογοι (Φάκελοι) </h3>
Οι κατάλογοι είναι χώροι που περιέχουν αρχεία και άλλους καταλόγους.
Αυτοί σας επιτρέπουν να οργανώσετε τα αρχεία σας σε διάφορες κατηγορίες.
Για παράδειγμα, μπορείτε να έχετε έναν κατάλογο με όλες τις φωτογραφίες σας και έναν άλλον κατάλογο με τα έγγραφα εργασίας σας.

<h3 style="text-align: left;"> Δομή Δέντρου </h3>
Τα αρχεία και οι κατάλογοι οργανώνονται σε μια δομή που ονομάζεται δομή δέντρου.
Στην κορυφή του δέντρου βρίσκεται ο "ρίζα" κατάλογος.
Κάθε κατάλογος μπορεί να περιέχει αρχεία και υποκατάλογους, δημιουργώντας μια ιεραρχία οργάνωσης.

Για παράδειγμα:
```
Ρίζα (Root)
|-- Φωτογραφίες
|   |-- Καλοκαίρι 2023
|   |-- Ταξίδι Παρίσι
|
|-- Έγγραφα
|   |-- Αναφορές
|   |-- Σημειώσεις
|
|-- Μουσική
|   |-- Αγαπημένα Τραγούδια
```

Με τη βοήθεια των αρχείων και των καταλόγων, μπορείτε να διατηρείτε τα δεδομένα σας οργανωμένα και εύκολα προσβάσιμα.

<h2 style="text-align: center;"> Επεκτάσεις Αρχείων </h2>

Οι επεκτάσεις αρχείων είναι κωδικοί που προσαρτώνται στα τέλη των ονομάτων αρχείων και δηλώνουν τον τύπο ή τη μορφή του περιεχομένου του αρχείου.
Αυτό βοηθά το λειτουργικό σύστημα και τις εφαρμογές να καταλάβουν πώς πρέπει να επεξεργαστούν ή να ανοίξουν ένα συγκεκριμένο αρχείο.
Ας ρίξουμε μια ματιά σε μερικές δημοφιλείς επεκτάσεις αρχείων και ορισμένες που αφορούν προγραμματισμό:

<h3 style="text-align: left;"> Γενικές Επεκτάσεις </h3>

* .txt (Κείμενο): Αρχείο κειμένου με απλό κείμενο.
* .docx, .pdf (Εγγράφου Κειμένου): Αρχεία εγγράφων που μπορεί να περιέχουν μορφοποιημένο κείμενο και εικόνες.
* .jpg, .png (Εικόνα): Εικόνες σε διάφορες μορφές.
* .mp3, .wav (Ήχος): Αρχεία ήχου.

<h3 style="text-align: left;"> Επεκτάσεις Προγραμματισμού </h3>

* .html (HyperText Markup Language): Χρησιμοποιείται για τη δημιουργία ιστοσελίδων.
* .css (Cascading Style Sheets): Χρησιμοποιείται για τον σχεδιασμό και την εμφάνιση ιστοσελίδων.
* .py (Python): Αρχεία κώδικα προγραμματισμού στη γλώσσα προγραμματισμού Python.
* .cpp (C++): Αρχεία κώδικα προγραμματισμού στη γλώσσα C++.
* .ino (Arduino): Αρχεία κώδικα για προγραμματισμό μικροελεγκτών Arduino.

Οι επεκτάσεις αρχείων είναι σημαντικές καθώς καθορίζουν πώς πρέπει να ερμηνευτεί και να ανοιχτεί το κάθε αρχείο,
ενώ και τα προγράμματα που θα χρησιμοποιηθούν για την επεξεργασία του.

<h2 style="text-align: center;"> Άνοιγμα Αρχείων </h2>

Για να ανοίξετε ένα αρχείο,
χρειάζεστε μια εφαρμογή που μπορεί να ερμηνεύσει τον τύπο του αρχείου.
Ανάλογα με τον τύπο του αρχείου,
χρησιμοποιούνται διάφορες εφαρμογές.
Ας δούμε μερικά παραδείγματα:

<h3 style="text-align: left;"> Κείμενο και Εγγράφου Κειμένου: </h3>

* Notepad (Windows): Μια βασική εφαρμογή κειμένου για τα Windows.
* Word (Microsoft Office): Ένα προηγμένο πρόγραμμα επεξεργασίας κειμένου.
* LibreOffice Writer: Ελεύθερο λογισμικό επεξεργασίας κειμένου, συμβατό με τα Microsoft Office αρχεία.

<h3 style="text-align: left;"> Εικόνες: </h3>

* Windows Photo Viewer (Windows): Προεπιλεγμένος προβολέας φωτογραφιών στα Windows.
* Photos (Windows 10): Η ενημερωμένη έκδοση του προβολέα φωτογραφιών στα Windows 10.
* Preview (Mac): Ο προβολέας φωτογραφιών που παρέχεται στο macOS.

<h3 style="text-align: left;"> Προγραμματισμός: </h3>

* Notepad++: Ένας απλός επεξεργαστής κειμένου για προγραμματιστές με πολλές λειτουργίες.
* Visual Studio Code (VSCode): Δημοφιλής και ευέλικτος επεξεργαστής κειμένου από τη Microsoft.
* JetBrains (π.χ. IntelliJ IDEA, PyCharm, WebStorm): Ολοκληρωμένα περιβάλλοντα ανάπτυξης για διάφορες γλώσσες προγραμματισμού.

Οι παραπάνω είναι μερικές από τις εφαρμογές που μπορείτε να χρησιμοποιήσετε για να ανοίξετε αρχεία σε διάφορους τύπους και κατηγορίες.
Επιλέξτε την κατάλληλη εφαρμογή ανάλογα με τον τύπο του αρχείου που θέλετε να ανοίξετε.
